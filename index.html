    Project Description

    Apjot


    Project Description

    Inspired by my love for writing, Apjot {A public journal of thoughts} is a blog website with fully integrated content management system for both Blog posts and Podcast episodes.

    It will serve as a home for my write ups and in the future a place for others to do same. Blog Posts are added dynamically from django database while Podcast episodes are embedded dynamically from Spotify. Both sections come with like and comment functionalities.


    Technologies used

    React,Css, Appwrite Api,Cloudinary (static file storage)


    Live website Github
